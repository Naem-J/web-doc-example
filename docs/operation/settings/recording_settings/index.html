An abridged web-based user guide

Recording Settings

To access Recording options, click Recording.
NOTE: Recording settings cannot be changed while recording video.

Basic and Advanced

To set basic settings, click Basic. To set advanced settings, click Advanced.

1 Movies Format buttons – Saves video files as .avi or .mov files.

2 Show Logo toggle switch – Displays the system logo in the bottom-left corner during operation if Record DataFusion is disabled.

3 Auto-Record toggle switch – Automatically starts recording when a case starts.

4 Record DataFusion toggle switch – Includes overlay information from a connected device.

5 Duration (min) toggle switch – Sets a maximum time limit (in minutes) for individual recordings.

6 Snapshots buttons – Saves snapshots as either .jpg or .png files.

7 Frame Rate slider – Sets the frames per second for recorded video. Higher frame rates increase file size.

8 Quality slider – Sets the recorded video quality. Raise the slider to improve quality and increase the file size or lower the slider to decrease quality and decrease file size.

9 Limit to 1080p toggle switch – Saves video in 1920x1080 resolution.

10 Raw Capture button – Opens the Raw Capture dialog box.

Raw Capture

This panel is intended for users conducting research projects. Capturing raw data from the system affects visualization. Utilize this function only with doctor approval.

1 Microscope information – Defines the microscope model, illumination percentage, and camera iris open percentage.

2 Additional method information – Enable the toggle switch for each method used in operation. If necessary, define the device model used and any additional requested information.

3 Notes field – Saves notes for the recording.
